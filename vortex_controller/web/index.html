How to connect
  1. Install Vortex on your device.
  2. Pick the Global network mode in the setup wizard.
  3. Paste the controller URL (auto-resolved via SNS) or an entry URL from this site.
  4. Verify the pubkey in your app matches the fingerprint above.
  5. You're connected. Everything else (node list, signatures) is handled automatically.
